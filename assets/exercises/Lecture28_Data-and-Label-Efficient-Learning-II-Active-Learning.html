Exercises

  1. Which of the following best describes the main goal of active learning?

    1. Maximize the size of the labeled dataset before training

    2. Achieve high model performance while minimizing labeling cost

    3. Eliminate the need for human annotation entirely

    4. Guarantee perfect classification after one training round

    Solution:
    b) Achieve high model performance while minimizing labeling cost

    Explanation:
    The main objective of active learning is to use a limited annotation budget efficiently. Instead of labeling all available data, the learner selectively queries the most informative samples so that strong model performance can be achieved with fewer labeled examples.

  2. What is the main difference between standard supervised learning and active learning?

    1. Supervised learning uses only unlabeled data, while active learning uses only labeled data

    2. Supervised learning assumes labels are already available, while active learning selectively requests labels during training

    3. Active learning does not require a machine learning model

    4. Supervised learning always requires fewer samples than active learning

    Solution:
    b) Supervised learning assumes labels are already available, while active learning selectively requests labels during training

    Explanation:
    In supervised learning, the labeled dataset is assumed to be prepared in advance. In active learning, the learner interacts with an unlabeled pool and chooses which samples should be labeled next under a limited budget.

  3. In the active learning pipeline, what is the role of the acquisition function?

    1. It provides the true labels for selected samples

    2. It ranks unlabeled samples according to their expected usefulness

    3. It replaces the training dataset with the test dataset

    4. It guarantees zero training error

    Solution:
    b) It ranks unlabeled samples according to their expected usefulness

    Explanation:
    The acquisition function is the mechanism that decides which unlabeled samples should be selected for annotation. It typically uses criteria such as uncertainty, diversity, or expected model improvement.

  4. Which metric is often more informative than accuracy for imbalanced classification problems?

    1. F1-score

    2. FLOPS

    3. Mean squared error

    4. IoU threshold

    Solution:
    a) F1-score

    Explanation:
    Accuracy may be misleading when one class dominates the dataset. The F1-score balances precision and recall, making it more appropriate for imbalanced classification settings.

  5. Why is the area under the performance curve (AUC) useful in active learning?

    1. It measures how much GPU memory is used during retraining

    2. It summarizes how efficiently performance improves as more labels are acquired

    3. It counts the number of oracle queries only

    4. It replaces all other evaluation metrics

    Solution:
    b) It summarizes how efficiently performance improves as more labels are acquired

    Explanation:
    In active learning, we care not only about final performance but also about how quickly the model improves with additional labeled samples. A higher AUC indicates a more label-efficient acquisition strategy.

  6. Entropy sampling selects samples that:

    1. Have the smallest feature norm

    2. Have the highest uncertainty across predicted class probabilities

    3. Are closest to the class centroid

    4. Have already been labeled

    Solution:
    b) Have the highest uncertainty across predicted class probabilities

    Explanation:
    Entropy measures uncertainty in the predicted class distribution. If the model spreads probability mass across several classes, the entropy is high, indicating that the sample is informative for annotation.

  7. What is the key idea behind margin sampling?

    1. Select samples with the largest distance from the training set

    2. Select samples with the smallest difference between the top two predicted class probabilities

    3. Select samples with the highest training loss only

    4. Select samples uniformly at random

    Solution:
    b) Select samples with the smallest difference between the top two predicted class probabilities

    Explanation:
    Margin sampling focuses on samples for which the model is nearly undecided between two classes. These are often near the decision boundary and are especially useful for refining the classifier.

  8. Which statement best describes diversity-based active learning?

    1. It selects samples that collectively cover the overall data distribution

    2. It always selects the most uncertain sample, even if redundant

    3. It ignores feature-space structure

    4. It is identical to least-confidence sampling

    Solution:
    a) It selects samples that collectively cover the overall data distribution

    Explanation:
    Diversity-based methods aim to avoid redundancy by selecting representative samples from different regions of the feature space. This improves global coverage of the dataset.

  9. Why can fusion-based active learning be useful?

    1. It combines diversity and difficulty, balancing dataset coverage and uncertainty

    2. It removes the need for oracle labeling

    3. It guarantees the lowest computational cost

    4. It always outperforms all other methods without tuning

    Solution:
    a) It combines diversity and difficulty, balancing dataset coverage and uncertainty

    Explanation:
    Fusion-based methods combine the strengths of diversity-based and difficulty-based strategies. This allows the learner to improve global representation early on while refining difficult decision boundaries later.

  10. What is a negative flip in the context of active learning deployment?

    1. A previously incorrect prediction becomes correct

    2. A previously correct prediction becomes incorrect after model updating

    3. The training loss becomes negative

    4. The acquisition function fails to assign a score

    Solution:
    b) A previously correct prediction becomes incorrect after model updating

    Explanation:
    A negative flip captures regression at the sample level. It occurs when the updated model makes an error on a sample that the previous model had classified correctly.

  11. What does the Negative Flip Rate (NFR) measure?

    1. The proportion of correctly classified samples in the test set

    2. The fraction of previously correct samples that become incorrect after model updating

    3. The number of labels acquired per active learning round

    4. The total annotation time for the dataset

    Solution:
    b) The fraction of previously correct samples that become incorrect after model updating

    Explanation:
    NFR is a deployment-oriented metric that measures prediction stability across model updates. It is useful because overall accuracy may improve even while some previously correct predictions are lost.
