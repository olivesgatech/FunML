Exercises

  1. Question 1: Which statement best describes the difference between k-means and GMM clustering?

    1. k-means performs soft clustering, while GMM performs hard clustering.

    2. k-means assigns each point to exactly one cluster, while GMM assigns posterior probabilities to clusters.

    3. k-means models each cluster with a full covariance matrix, while GMM uses only centroids.

    4. There is no difference; both methods are equivalent in general.

    Solution: The correct answer is (b). k-means is a hard-clustering method that assigns each sample to exactly one cluster, whereas GMM is a soft-clustering method that computes posterior probabilities (responsibilities) for each cluster.

  2. Question 2: In the EM algorithm for GMMs, what is computed during the E-step?

    1. Updated cluster means and covariance matrices

    2. Posterior responsibilities for each data point under each component

    3. The optimal number of clusters

    4. A hard assignment of each point to its nearest centroid

    Solution: The correct answer is (b). In the E-step, the algorithm computes the responsibilities \[\gamma_i^j = p(z_i=j \mid x_i),\] which represent the posterior probability that sample \(x_i\) was generated by component \(j\).

  3. Question 3: Why is the LAB color space often preferred over RGB for clustering-based image segmentation?

    1. LAB always reduces the number of segments.

    2. LAB separates luminance and chrominance, making perceptual similarity more meaningful for clustering.

    3. LAB guarantees perfect segmentation.

    4. RGB cannot be used for clustering.

    Solution: The correct answer is (b). LAB is often preferred because it separates luminance from chrominance, which better reflects perceptual similarity and can produce more meaningful clusters for segmentation.
