Causal mask zeroes out the upper triangle of QK^T before softmax so a token at position t cannot attend to future tokens t+1, ..., n.

Step 3 — Softmax Attention Weights and Output O = A.V

Attention weights A = softmax(QK^T / sqrt(d_k))
Output O = A . V (n x d_k)

Per-row property: each row of A is a probability distribution summing to 1 (over allowed past positions). Row t of O is the value-weighted summary that token t will use downstream.

Modern LLMs (GPT-4, Claude) support 100K-1M token contexts; KV cache becomes the dominant memory bottleneck during inference.

Why it matters: attention compute grows as O(n^2 . d) while the per-layer projection cost is only O(n . d^2). For long contexts, attention dominates — and the KV cache (n . d_k floats per layer per head) explodes.

Autoregressive Generation: Without vs With KV Cache

Each step appends one new token. Without cache, K and V are recomputed for the entire sequence. With cache, only the new row is computed; prior K/V rows are reused.
